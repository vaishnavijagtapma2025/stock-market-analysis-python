# Starter Idea - Swachh Bharat And Toilet Access

## Why This Is Interesting

This topic gives you a clear policy setting, a visible outcome, and a natural before-versus-after structure. It can become a good causal-style project if you keep the claim modest and the design disciplined.

## Likely Project Type

Causal

## Core Question

Did toilet access improve more in some places after Swachh Bharat than in comparable places with weaker early exposure?

## Good Stakeholder

A sanitation mission team, rural development department, or public policy researcher.

## Likely Main Metric

Your main metric could be:

- a difference-in-differences estimate in percentage points
- or a simpler before-versus-after gap if you later decide the design should stay descriptive

## Simple Baseline

Start with:

- plain before-versus-after change
- or treated-versus-comparison difference without controls

## Likely Data Sources

- NFHS-4
- NFHS-5
- public administrative summaries if needed for project context

## First 24-Hour Feasibility Check

- confirm that your toilet-access variable is defined comparably across survey rounds
- decide what makes a place "more exposed" versus "less exposed"
- build one tiny comparison table

## Main Risk

The causal design can become weak if treatment and comparison groups are vague. If that happens, narrow the claim early or shift the project to a descriptive design with approval.

## How To Keep It Small

Start with:

- one outcome
- one treatment rule
- one clean comparison design

Small and credible is much better than ambitious and shaky.
